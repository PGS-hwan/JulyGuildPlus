# 🔐 权限说明

## 🎯 基础权限

| 权限节点 | 说明 |
|----------|------|
| `JulyGuildPlus.use` | 使用插件基本功能（打开各类 GUI） |
| `JulyGuildPlus.create.colored` | 创建公会时可使用彩色代码（如 `&a`, `&c`） |

## 🔧 管理权限

| 权限节点 | 说明 |
|----------|------|
| `JulyGuildPlus.admin` | 插件管理权限（重载、调试等） |

## 📝 权限配置示例

```yaml
# LuckPerms 示例
groups:
  default:
    permissions:
      - JulyGuildPlus.use
  admin:
    permissions:
      - JulyGuildPlus.admin
      - JulyGuildPlus.create.colored
```
> 💡 提示: 若不配置权限，默认玩家拥有 JulyGuildPlus.use 权限
