# ❓ 常见问题

## Vault Hook 失败

**现象**: 控制台提示 `[JulyGuildPlus] Vault: Hook失败, 插件将被卸载!`

**解决步骤**:
1. ✅ 确认已安装 **Vault** 插件
2. ✅ 确认已安装并启用**经济插件**（如 Essentials）
3. ✅ 检查经济插件是否正常工作（使用 `/eco` 测试）
4. ✅ 重启服务器（完全重启，非重载）
5. ✅ 检查 Vault 是否为最新版本

---

## GUI 内变量不显示

**现象**: `%vault_eco_balance%` 等变量原样输出，不解析

**解决步骤**:
1. ✅ 确认配置中 `use_papi: true`
2. ✅ 安装 **PlaceholderAPI** 插件
3. ✅ 下载并加载对应扩展：

   ```bash
   /papi ecloud download Player
   /papi ecloud download Vault
   /papi ecloud download PlayerPoints
   ```

4. ✅ 执行 /papi reload 或重启服务器

## 玩家名显示为 UUID

**现象**: GUI 中会长栏显示为 `a97811d6-e2ca-3171-...` 而非玩家名

**原因**: 使用了旧插件，新插件无此问题

**解决方案**: 迁移至新插件

## 配置重载无效

**现象**: 修改配置后 /jgp reload 未生效

**解决步骤**:
1. ✅ 检查配置文件编码是否为 UTF-8
2. ✅ 检查 YAML 语法是否正确（使用在线校验工具）
3. ✅ 确认修改的是正确文件路径
4. ✅ 重启服务器尝试

## 公会商店无法购买

**现象**: 点击商品无反应或提示错误

**排查步骤**:
1. ✅ 检查 GuildShop.yml 配置格式
2. ✅ 确认公会币足够
3. ✅ 检查 reward_type 是否有效
4. ✅ 查看控制台是否有错误日志
