# 🎮 指令列表

## 🎯 玩家指令

| 指令 | 说明 | 所需权限 |
|------|------|----------|
| `/jgp gui` | 打开公会主界面 | `JulyGuildPlus.use` |
| `/jgp version` | 显示插件版本信息 | 无 |

## 🔧 管理指令

| 指令 | 说明 | 所需权限 |
|------|------|----------|
| `/jgp reload` | 重载插件配置 | `JulyGuildPlus.admin` |
| `/jgp admin debug` | 切换日志级别（DEBUG/INFO） | `JulyGuildPlus.admin` |
| `/jgp admin getItemInfo` | 获取手上物品的材质和数据值 | `JulyGuildPlus.admin` |

## 📌 指令别名

- `/guild` - 主指令别名
- `/jg` - 主指令别名
- `/gh` - 主指令别名（“公会”的拼音缩写）

> **💡 提示**: 权限节点区分大小写
