# 📥 安装指南

## ✅ 前置要求

- **Java 8** 或更高版本
- **Bukkit/Spigot** 1.7.10 ~ 1.21.11
- **Vault** 插件（必须）
- 支持的经济插件（如 PlayerPoints）

## 📦 所需文件

**插件本体**: JulyGuildPlus_x.x.x.jar

>💡 提示：无需下载 JulyLibrary，插件内部已集成

## 📝 安装步骤

1. **下载文件**: 获取插件本体
2. **放入插件文件夹**: 将`.jar` 文件放入服务端 `plugins` 文件夹
3. **检查 Vault**: 确保已安装 Vault 插件且能成功 Hook
4. **重启服务器**: 完全重启服务器
5. **验证安装**: 检查控制台输出
- 成功安装后，控制台应显示类似信息：

```bash
[JulyGuildPlus] 插件版本: x.x.x
[JulyGuildPlus] bStats统计: 已启用
[JulyGuildPlus] PlaceholderAPI: Hook成功!
[JulyGuildPlus] Vault: Hook成功!

```
