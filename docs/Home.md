# JulyGuildPlus Wiki

![JulyGuildPlus](https://github.com/PGS-hwan/JulyGuildPlus/blob/main/docs/assets/JulyGuildPlus.png)

欢迎来到 **JulyGuildPlus** 插件文档。这是一个功能强大的 Minecraft 公会插件，支持多种经济插件和丰富的自定义选项。

## 📋 支持版本

- **游戏版本**: Bukkit/Spigot 1.7.10 ~ 1.21.11
- **Java 版本**: Java 8+
- **依赖插件**: Vault
